To generate updated documentation files, all you have to do is:

1. Download and install DoxyWizard from http://www.doxygen.nl/download.html
2. Clone PID-UdG repository from https://github.com/EnricRG/PID-UdG
3. Open DoxyWizard and load the doxygen config file inside PID-UdG/docs/doxygen called DoxygenConfig
4. Set PID-UdG/docs/doxygen as running directory
5. Go to 'Run' tab and press 'Run Doxygen'

You'll find all your html files inside PID-UdG/docs/output
